# Dread Delusion — Русификатор 1.3.3

Русский перевод игры **Dread Delusion** для актуальной версии **1.3.3** (Steam, Unity 2022.3 / IL2CPP).

- **Базовый перевод:** LiquidRing — портирован на 1.3.3 (оригинал был под старую 2019.4-сборку).
- **Новый контент 1.3.3** (меню, настройки, интро, журнал квестов, предметы, книги) добит автоматически.
- **Шрифт** игры (alagard) с кириллицей подаётся в рантайме через BepInEx — родной блэклеттер-стиль, без квадратов.

## Скачать

→ **[Releases](../../releases/latest)** — качай `DreadDelusion_RUS_1.3.3_by_rollacode.zip`.

## Установка

1. Распакуй архив в `...\Steam\steamapps\common\Dread Delusion\windows_content\` с заменой файлов.
2. Запусти игру через Steam. Всё сразу на русском.

Подробности (откат, что делать после апдейта Steam) — в `ЧИТАЙ_МЕНЯ.txt` внутри архива.

## Как это сделано

- Текст вписан напрямую в TextAsset'ы внутри `resources.assets` (матчинг EN↔RU по ключам диалогов/таблиц).
- Шрифт: alagard-unicode SDF грузится плагином BepInEx 6 IL2CPP и регистрируется как глобальный TMP-фолбэк.

Порт на 1.3.3 + шрифт-плагин + сборка: **rollacode**. Базовый перевод: **LiquidRing**.
